# Campaign Pathing (WIP)
<div class="note-block">

Every bullet point should be considered a "chain of events" that mostly ends in a logout. The goal is to go through every chain in sequence to minimize the number of times you have to be in town. Check Legend below for all points of interests.

</div>

### Legend : [WP = Waypoints](gemblue) [Logout](gemred) [Trials](trial) [Quest Item](gemquest)

---
## Act 1
<ul class="custom-list bullet-list">

- Coast Kill <a href="npc">Hillock</a> → Vendor Check 1 → Mud Flats → [Submerged WP](gemblue) → Coast → Tidal Island → Kill <a href="npc">Hailrake</a> → [Logout](gemred)
- [Submerged WP](gemblue) → Drop TP @ Bridge for Dweller → Ledge (Take goat path for XP) → Climb → [Lower Prison WP](gemblue) → WP to A1 Town → Take TP → Kill <a href="npc">Dweller</a> → [Logout](gemred)
- Lower Prison [Trial 1](trial)> Upper Prison → Kill <a href="npc">Brutus</a> → [Logout](gemred)
- Vendor Check 2 → Prisoners Gate → [Ship Graveyard WP](gemblue) → TP @ Ship Graveyard Cave → [Caverns of Wrath WP](gemblue) → WP to A1 Town → Take TP → [Ship Graveyard Cave (Allflame)](gemquest) → Kill <a href="npc">Fairgraves</a> → [Logout](gemred)
- Vendor Check 3 → [Caverns of Wrath WP](gemblue) → Caverns of Anger → Kill <a href="npc">Mervail</a> → [Logout](gemred)

</ul>

---
## Act 2
<ul class="custom-list bullet-list">

- [Southern Forest WP](gemblue) (hug right wall) → A2 Town → Vendor Check 1 → Old Fields → [Crossroads WP](gemblue) → Chamber 1 → Chamber 2 [Trial 2](trial) → [Kill Fidelitas](gemquest) → [Logout](gemred)
- [Crossroads WP](gemblue)  Fellshrine Ruins → The Crypt 1 [Trial 3](trial) → The Crypt 2 [Golden Handjob](gemquest) → [Logout](gemred)
- Vendor Check 2 → Riverways → [Western Forest WP](gemblue) → Run opposite of WP → Weaver's Chambers → [Kill Weaver](gemquest) → [Logout](gemred)
- Vendor Check 3 → Crossroads → Broken Bridge → [Kill Kraitlyn](gemquest) → [Logout](gemred)
- Riverways → Wetlands [Kill Oak](gemquest) → [Wetlands WP](gemblue) → [Western Forest WP](gemblue) → Run same side of WP → [Kill Alira](gemquest) → [Kill Generals](gemquest) → Run down through the door → TP to A1 Town → Take Reward → [Logout](gemred)
- WP to A2 Town → [Take Apex](gemquest) → [Wetlands WP](gemblue) → Vaal Ruins → Click ball → Northern Forest → Caverns → Ancient Pyramid → Kill <a href="npc">Oversoul</a> → [Logout](gemred)

</ul>



## Act 3
<ul class="custom-list bullet-list">

- Vendor Check 1 → City of Sarn WP → Defeat Guards → Talk to Clarissa → A3 Town → The Slums → Drop TP @ Sewer Grate → Crematorium [Trial 4](trial) → Kill Peity → Suicide → Resurrect in Town → Vendor Check 2 → Talk to Clarissa (Take Keys) → Enter your TP → The Sewers → Marketplace WP → The Catacombs [Trial 5](trial) → [Logout](gemred)
- Marketplace WP → Battlefront WP → If lvl 24 go Docks, if Less than 24 go Solaris 1; I personally try to hit as close to 24 by battlefront so I can go Docks
- Docks (find gemquest item) → [Logout](gemred)
- Vendor Check 3 (you should be 24, so gamble at Maramoa for 4-Link) → Battlefront WP → Solaris 1 → Solaris 2 WP Take Dialla reward → [Logout](gemred)
- Sewers → Butthole → Ebony WP → Kill Gravicius if you need gems → Lunaris 1 → Lunaris 2 (Kill Peity take gemquest item) → [Logout](gemred)
- Ebony WP → Imperial Gardens WP [Trial 6](trial) → Library if mandatory otherwise skip → Sceptre of God → Upper Sceptre → Kill Dominus → [Logout](gemred)

</ul>

## Act 4
<ul class="custom-list bullet-list">

- Aqueducts → A3 Town → Dried Lake → [Kill Voll](gemquest) [Logout](gemred)
- Mines 1 → Mines 2 [Find Deshret](gemquest) → [Crystal Veins WP](gemblue) → [Trial of Ascendancy 1](trial) → [Logout](gemred)
- [Crystal Veins WP](gemblue) → Daresso's Dream → Grand Arena → [Kill Daresso](gemquest) → [Logout](gemred)
- [Crystal Veins WP](gemblue) → Kaom's Dream → Kaom's Stronghold → [Kill Kaom](gemquest) → [Logout](gemred)
- [Crystal Veins WP](gemblue) → Belly 1 → Belly 2 → Kill <a href="npc">Piety</a> → Harvest → Kill <a href="npc">Malachai</a> → [Logout](gemred)

</ul>
