# Resources
Updated: Feb 11, 2026

<div class="note-block">
This might seem like a long list but I practically use every single one of these multiple times a league. Some tools/websites I probably could not give up when playing POE1.
</div>

## <span id="3rdparty"></span>Most Used 3rd Party Tools
<ul class="custom-list bullet-list">

- <a href="https://pathofbuilding.community/">Path of Building</a>: This is the real game
- <a href="https://github.com/SnosMe/awakened-poe-trade">Awakened POE Trade</a>: Defactor POE1 price checker, can also make useful chat macros.
- <a href="https://www.filterblade.xyz/">Filterblade</a>: Defacto loot filter generator. Highly customizable, just use this, I've tried them all this is straight up the GOAT.
- <a href="https://github.com/ChaosRecipeEnhancer/ChaosRecipeEnhancer">Chaos Recipe Enhancer</a>: Chaos recipe tool. Highlights items in stash. Incredibly useful for league start.

</ul>

## <span id="browser"></span>Browser Extensions
<ul class="custom-list bullet-list">

- <a href="https://chromewebstore.google.com/detail/better-pathofexile-tradin/fhlinfpmdlijegjlpgedcmglkakaghnk">Better POE Trading</a>: Basically a bookmark system for all your complex trade searches
- <a href="https://chromewebstore.google.com/detail/path-of-exile-trade-fuzzy/mkbkmkampdnnbehdldipgjhbablkmfba?hl=en">POE Fuzzy Search</a>: This adds "~" in the search boxes, makes it MUCH faster to find mods and items
  
</ul>

## <span id="websites"></span>Most Used Websites
<ul class="custom-list bullet-list">

- <a href="https://poe.ninja/poe1/builds">PoE.Ninja</a>: Single best place to research builds and market prices.</li>
- <a href="https://poe.watch">PoE.Watch</a>: Market price tracker similar to ninja.</li>
- <a href="https://poedb.tw/us/Modifiers/">PoEDB</a>: Very useful for finding modifiers and their tiers on every single item.</li>
- <a href="https://wealthyexile.com/">WealthyExile</a>: Currency tracker. Great way to see if you're a shit mapper or blaster</li> 
- <a href="https://www.craftofexile.com/">Craft of Exile</a>: Similar to PoEDB but different UI. Also has craft simulator thats very useful.</li> 
- <a href="https://poeplanner.com/atlas-tree">POEPlanner</a>: Plan your Atlas tree or Passive tree in browser.</li> 
- <a href="https://www.poe.re/">Path of Regex</a>: Best website to make REGEX.</li> 
- <a href="https://disenchant.easyexile.tools/">Disenchant Tool</a>: Find which items are good to disenchant for Kingsmarch.</li> 
- <a href="https://vilsol.github.io/timeless-jewels">Timeless Calculator</a>: Find timeless jewels.</li> 
- <a href="https://siveran.github.io/calc.html">Vorici Calculator</a>: Find if its cheaper to chrome or to use jeweller orb to color/socket.</li> 
- <a href="https://theodorejbieber.github.io/PoEClusterJewelCalculator/">Cluster Jewel Calculator</a>: Easy way to find Large clusters where you don't take the middle notable.</li>
- <a href="https://hideoutshowcase.com/">Hideout Showcase</a>: Find ways to spend money on this game.</li>

</ul>

---

## <span id="creator"></span>Good Creator Content
<ul class="custom-list bullet-list">

- <a href="https://www.youtube.com/@TeamFocusPOE/videos">Team Focus YouTube</a> - Arguably the best videos for learning group speedrunning.
  - Their <a href="https://www.youtube.com/watch?v=A-jef7XshtU">Group Running POE 101</a> and <a href="https://www.youtube.com/watch?v=XA3_xwJgWe0">Routing a Run</a> videos are the big standouts.
- <a href="https://www.youtube.com/playlist?list=PL1sXxxK5qcP1DFMN4Blg58TAAOwNTsvc9">LLYD</a> - 5-10 min Act 1 Layout tutorials
- <a href="https://www.definitivguide.com/docs/PoE1/layout">Cyclon</a> - Excellent layout breakdowns of every zone. Would recommend if you want to get faster at campaign
- <a href="https://www.youtube.com/@Tuadh-PoE/videos">Tuadh </a>- Posts a ton of 2 stone run vids. Incredible resource for league start.


</ul>
