Campaign Pathing

Every bullet point should be considered a "chain of events" that mostly ends in a logout. The goal is to go through every chain in sequence to minimize the number of times you have to be in town. Check Legend below for all points of interests.

Legend : WP = Waypoints Logout Trials Quest Item

Act 1

    • Coast Kill Hillock → Vendor Check 1 → Mud Flats → Submerged WP → Coast → Tidal Island → Kill Hailrake → Logout
    • Submerged WP → Drop TP @ Bridge for Dweller → Ledge (Take goat path for XP) → Climb → Lower Prison WP → WP to A1 Town → Take TP → Kill Dweller → Logout
    • Lower Prison Trial 1> Upper Prison → Kill Brutus → Logout
    • Vendor Check 2 → Prisoners Gate → Ship Graveyard WP → TP @ Ship Graveyard Cave → Caverns of Wrath WP → WP to A1 Town → Take TP → Ship Graveyard Cave (Allflame) → Kill Fairgraves → Logout
    • Vendor Check 3 → Caverns of Wrath WP → Caverns of Anger → Kill Mervail → Logout

Act 2

    • Southern Forest WP (hug right wall) → A2 Town → Vendor Check 1 → Old Fields → Crossroads WP → Chamber 1 → Chamber 2 Trial 2 → Kill Fidelitas → Logout
    • Crossroads WP Fellshrine Ruins → The Crypt 1 Trial 3 → The Crypt 2 Golden Handjob → Logout
    • Vendor Check 2 → Riverways → Western Forest WP → Run opposite of WP → Weaver's Chambers → Kill Weaver → Logout
    • Vendor Check 3 → Crossroads → Broken Bridge → Kill Kraitlyn → Logout
    • Riverways → Wetlands Kill Oak → Wetlands WP → Western Forest WP → Run same side of WP → Kill Alira → Kill Generals → Run down through the door → TP to A1 Town → Take Reward → Logout
    • WP to A2 Town → Take Apex → Wetlands WP → Vaal Ruins → Click ball → Northern Forest → Caverns → Ancient Pyramid → Kill Oversoul → Logout

Act 3

    • Vendor Check 1 → City of Sarn WP → Defeat Guards → Talk to Clarissa → A3 Town → The Slums → Drop TP @ Sewer Grate → Crematorium Trial 4 → Kill Peity → Suicide → Resurrect in Town → Vendor Check 2 → Talk to Clarissa (Take Keys) → Enter your TP → The Sewers → Marketplace WP → The Catacombs Trial 5 → Logout
    • Marketplace WP → Battlefront WP → If lvl 24 go Docks, if Less than 24 go Solaris 1; I personally try to hit as close to 24 by battlefront so I can go Docks
    • Docks (find gemquest item) → Logout
    • Vendor Check 3 (you should be 24, so gamble at Maramoa for 4-Link) → Battlefront WP → Solaris 1 → Solaris 2 WP Take Dialla reward → Logout
    • Sewers → Butthole → Ebony WP → Kill Gravicius if you need gems → Lunaris 1 → Lunaris 2 (Kill Peity take gemquest item) → Logout
    • Ebony WP → Imperial Gardens WP Trial 6 → Library if mandatory otherwise skip → Sceptre of God → Upper Sceptre → Kill Dominus → Logout

Act 4

    • Aqueducts → A3 Town → Dried Lake → Kill Voll Logout
    • Mines 1 → Mines 2 Find Deshret → Crystal Veins WP → Trial of Ascendancy 1 → Logout
    • Crystal Veins WP → Daresso's Dream → Grand Arena → Kill Daresso → Logout
    • Crystal Veins WP → Kaom's Dream → Kaom's Stronghold → Kill Kaom → Logout
    • Crystal Veins WP → Belly 1 → Belly 2 → Kill Piety → Harvest → Kill Malachai → Logout
